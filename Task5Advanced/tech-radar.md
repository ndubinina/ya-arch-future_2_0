# Технический радар

## Adopt

| Категория       | Технология / подход       | Комментарий                                                                            |
| --------------- | ------------------------- |----------------------------------------------------------------------------------------|
| Архитектура     | Event-Driven Architecture | Независимое развитие, гибкое масштабирование                                           |
| Архитектура     | Data Mesh                 | Независимое развитие доменов и данных, зона ответственности команд знакомых с бизнесом |
| Архитектура     | Self-Service BI           | Самостоятельная работа пользователей с данными дает гибкость                           |
| Интеграции      | Kafka                     | Основа событийной платформы                                                            |
| Хранение данных | S3 Object Storage         | Масштабируемое облачное хранилище для сырых данных                                     |

| Аналитика       | Trino                     | Единая точка SQL-доступа к данным                                                      |
| Метаданные      | DataHub                   | Каталог данных и lineage                                                               |
| BI              | Apache Superset           | Портал самообслуживания и аналитика                                                    |
| IaC             | Terraform                 | Управление инфраструктурой как кодом                                                   |
| CI/CD           | GitLab CI                 | Автоматизация поставки инфраструктуры                                                  |
